# ATM System (CLI Based)

This is a simple ATM system project developed in C++ using a Command Line Interface (CLI). It simulates basic banking operations through a menu-driven program.

## Features:
- Check Balance
- Deposit Money
- Withdraw Money
- Exit System

## Menu Options:
1. Check Balance  
2. Deposit Balance  
3. Withdraw Balance  
4. Exit  

## Description:
This project is a CLI-based application where users interact through the terminal. It uses a menu-driven approach to perform basic ATM operations.

## Concepts Used:
- Functions
- Conditional Statements (if-else / switch)
- Loops
- Basic Input/Output

This project helps in understanding how real-world systems can be implemented using simple programming logic in C++.

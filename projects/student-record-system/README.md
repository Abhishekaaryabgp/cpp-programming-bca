# Student Record Management System (File Handling - CLI)

This is a Student Record Management System developed in C++ using file handling. It allows users to manage student data through a menu-driven Command Line Interface (CLI).

## Features:
- Register a new student
- Show all student records
- Search student record
- Update student record
- Delete student record
- Exit system

## Menu Interface:
=============================================================
                STUDENT MANAGEMENT SYSTEM
=============================================================
1. Register a Student
2. Show All Records
3. Search Record
4. Delete Record
5. Update Record
6. Exit
=============================================================

## Description:
This project uses file handling to store and manage student data permanently. Users can perform CRUD operations (Create, Read, Update, Delete) on student records.

## Concepts Used:
- File Handling (fstream)
- Structures or Classes
- Functions
- Conditional Statements
- Loops
- Menu-driven programming

## Purpose:
- Understand persistent data storage
- Learn real-world application logic
- Practice CRUD operations in C++

This project demonstrates how data can be stored, retrieved, and modified using file handling in C++.

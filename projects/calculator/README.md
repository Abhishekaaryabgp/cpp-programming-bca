# Calculator (CLI)

This is an advanced calculator application developed in C++ using a Command Line Interface (CLI). It allows users to perform multiple arithmetic operations in a loop-based system.

## Features:
- Addition
- Subtraction
- Multiplication
- Division
- Modulus
- Continuous calculation (no restart required)
- Exit option

## Description:
This calculator uses a menu-driven approach where users can repeatedly perform operations until they choose to exit. It improves user interaction compared to basic single-run programs.

## Concepts Used:
- Loops (while loop)
- Conditional Statements (switch-case)
- Functions (optional extension)
- Input/Output handling
- Basic error handling (division by zero)

## How to Run:
1. Compile the program
2. Run the executable
3. Select operation from menu
4. Enter numbers and get result

This project demonstrates how basic arithmetic logic can be extended into a more user-friendly and reusable application.

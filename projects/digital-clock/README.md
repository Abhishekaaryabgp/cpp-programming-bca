# Digital Clock (CLI)

This is a digital clock application developed in C++ using a Command Line Interface (CLI). It displays time in hours, minutes, and seconds format and updates every second.

## Features:
- Displays time in HH:MM:SS format
- Real-time clock simulation
- AM/PM format display
- Continuous running clock

## Description:
This project simulates a digital clock using loops and time delay functions. The program updates the time every second and displays it dynamically in the terminal.

## Concepts Used:
- Loops (nested loops)
- Conditional statements
- Time delay using sleep
- Input/Output handling

## How to Run:
1. Compile the program
2. Run the executable
3. Enter initial time (HH MM SS)
4. Watch the clock run continuously

This project helps in understanding time-based logic and real-time simulation in C++.

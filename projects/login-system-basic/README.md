# Basic Login System (CLI)

This is a basic login system developed in C++ using a Command Line Interface (CLI). It verifies user credentials using predefined username and password.

## Features:
- User login authentication
- Username and password verification
- Limited login attempts
- Simple and easy interface

## Description:
This project checks user-entered credentials against predefined values. It allows multiple attempts for login and displays success or failure messages.

## Concepts Used:
- Classes and Objects
- Conditional Statements (if-else)
- Loops
- String handling

## Purpose:
- Understand basic authentication logic
- Learn how login systems work
- Practice condition-based programming

This project is a beginner-level implementation of a login system in C++.
